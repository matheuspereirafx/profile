## 📖 About the Project

This is a static personal profile page developed entirely from scratch. The project was created as one of the core practical activities of the **Le Wagon Web Development Bootcamp**.

The interface was designed to be minimalist, direct, and professional, applying practical **UI/UX design** and visual structuring concepts to create an engaging presentation for recruiters and visitors.

---

## 🎯 Learning Objectives

The primary focus of this project was not to use frameworks or ready-made libraries, but to return to the roots of the web to deepen mastery of structure and styling. Key practice areas included:

* **Semantic HTML:** Correct use of tags (`<header>`, `<main>`, `<section>`, etc.) to improve SEO and screen reader accessibility.
* **Modern CSS:** Mastery of essential layout properties like **Flexbox** and **CSS Grid**, avoiding unnecessary absolute positioning.
* **Mobile First & Responsiveness:** Utilizing *Media Queries* to ensure the layout adapts fluidly from smartphones to ultrawide monitors.
* **Clean Code:** Organizing classes, CSS variables for color palettes, and typography to keep style files clean and maintainable.
