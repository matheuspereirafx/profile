## 📖 Sobre o Projeto

Esta é uma página de perfil pessoal estática, desenvolvida do zero. O projeto nasceu como uma das atividades práticas fundamentais do **Bootcamp de Web Development da Le Wagon**. 

A interface foi projetada para ser minimalista, direta e profissional, aplicando conceitos práticos de **UI/UX design** e estruturação visual, visando criar uma apresentação agradável para recrutadores e visitantes.

---

## 🎯 Objetivos de Aprendizado

O principal foco deste projeto não foi utilizar frameworks ou bibliotecas prontas, mas sim voltar às raízes da web para aprofundar o domínio estrutural e de estilização. Os principais pontos de prática foram:

* **HTML Semântico:** Uso correto de tags (`<header>`, `<main>`, `<section>`, etc.) para melhorar o SEO e a acessibilidade para leitores de tela.
* **CSS Moderno:** Domínio de propriedades essenciais de layout, como **Flexbox** e **CSS Grid**, evitando posicionamentos absolutos desnecessários.
* **Mobile First & Responsividade:** Utilização de *Media Queries* para garantir que o layout se adapte fluidamente desde telas de smartphones até monitores ultrawide.
* **Clean Code:** Organização de classes, variáveis CSS para paleta de cores e tipografia, mantendo o arquivo de estilos limpo e fácil de manter.

## 🚀 Estrutura e Funcionalidades

| Seção | Descrição |
| :--- | :--- |
| 👤 **Apresentação** | Resumo profissional, foto de perfil e título de atuação. |
